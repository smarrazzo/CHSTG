# Collecter la documentation de démontage

|ID          |
|------------|
|CHSTG-INFO-02|

## Résumé

Ce contrôle consiste à collecter la documentation disponible publiquement décrivant comment l'appareil peut être démonté. L'objectif est de préparer les futures étapes d'analyse matérielle en comprenant la procédure d'ouverture sans effectuer aucune action physique sur l'appareil.

## Objectifs du test
- Identifier les procédures de démontage officielles ou non officielles
- Déterminer les méthodes d'ouverture et les points d'accès
- Identifier les outils nécessaires
- Anticiper les composants fragiles ou les risques lors du démontage
- Identifier les mécanismes de protection potentiels (scellés anti-effraction, interrupteurs d'intrusion, coupe-circuit ou contre-mesures similaires)

## Comment tester
1. Utiliser le modèle d'appareil exact identifié précédemment.

2. Effectuer des recherches en ligne en combinant le modèle avec des mots-clés tels que :
   - disassembly (démontage)
   - teardown
   - repair (réparation)
   - service manual (manuel de service)

3. Prioriser les plateformes vidéo et les sources de documentation.

Exemples de recherches :
`hp 440 g7 disassembly`
![hp 440 g7 disassembly](img/youtube.jpg)

Cela peut fournir des vidéos explicatives, notamment des vidéos de réparation non officielles et des vidéos officielles du fabricant :
- https://www.youtube.com/watch?v=vmipNT6FMMA
- https://www.youtube.com/watch?v=2FG1abY5OYg

4. Examiner le matériel collecté pour identifier :
   - La séquence d'ouverture
   - Les vis cachées ou les clips
   - La présence de scellés anti-effraction ou d'autocollants
   - Les interrupteurs ou capteurs de détection d'intrusion

5. Collecter et archiver les ressources pertinentes pour une utilisation ultérieure lors de l'analyse matérielle.
