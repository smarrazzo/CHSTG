# Collecter la documentation Board-View

|ID          |
|------------|
|CHSTG-INFO-04|

## Résumé

Ce contrôle vise à localiser la documentation boardview de la carte mère en utilisant la référence de carte exacte identifiée précédemment. L'objectif est d'identifier l'existence et la disponibilité des fichiers de disposition physique de la carte mère pour une analyse matérielle ultérieure sans effectuer aucune interaction physique avec l'appareil.

## Objectifs du test
- Identifier la documentation boardview de la carte mère
- Déterminer la disponibilité des fichiers de disposition physique de la carte
- Identifier les sources donnant accès aux fichiers boardview

## Comment tester
1. Utiliser la référence carte mère identifiée lors du test CHSTG-INFO-01.

2. Effectuer des recherches en ligne en combinant la référence de carte avec des mots-clés tels que :
   - boardview
   - board view
   - motherboard boardview
   - pcb layout (disposition PCB)

Exemples de recherche (référence issue du test CHSTG-INFO-01) :

- `DAXK9FMB6C0 boardview`
  ![DAXK9FMB6C0 boardview](img/123.png)

- `DA0X8IMB8E0 boardview`
  ![DA0X8IMB8E0 boardview](img/00071854.png)

- `NB6099C_PCB_MB_MP1.0 boardview`  
  Aucun résultat trouvé (illustrant l’absence de documentation schématique publique pour certains modèles).


3. Examiner les résultats de recherche pour déterminer si la documentation boardview existe.

4. Identifier le type d'accès requis :
   - Fichiers accessibles publiquement
   - Téléchargement payant
   - Plateformes d'accès restreint ou VIP

5. Documenter la disponibilité et les conditions d'accès pour une utilisation ultérieure.
