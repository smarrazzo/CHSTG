# Collecter les images de micrologiciel

|ID          |
|------------|
|CHSTG-INFO-05|

## Résumé

Ce contrôle vise à identifier et collecter les images de micrologiciel disponibles et leurs versions en utilisant la référence de l'appareil sur le site web du fabricant. L'objectif est de préparer les étapes d'analyse ultérieures en connaissant quelles versions de micrologiciel existent pour l'appareil sans interagir physiquement avec le système.

## Objectifs du test
- Identifier les versions de micrologiciel disponibles pour l'appareil
- Localiser les images de micrologiciel officielles
- Déterminer l'historique des mises à jour du micrologiciel

## Comment tester
1. Utiliser le modèle d'appareil exact identifié lors du test CHSTG-INFO-01.

2. Accéder au site d'assistance officiel du fabricant.

3. Rechercher l'appareil en utilisant :
   - Le nom du modèle
   - Le numéro de produit
   - Le numéro de série ou d'étiquette de service (si pris en charge)

4. Naviguer vers la section téléchargement du micrologiciel ou du BIOS.

5. Identifier et noter :
   - Les versions de micrologiciel disponibles
   - Les dates de publication
   - Les images de micrologiciel téléchargeables

6. Archiver les informations pour une analyse ultérieure.

## Remédiation
Non applicable.
