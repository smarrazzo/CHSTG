# Collecter les fiches techniques des puces

|ID          |
|------------|
|CHSTG-INFO-08|

## Résumé

Ce contrôle vise à collecter la documentation technique (fiches techniques) des composants identifiés précédemment. L'objectif est d'obtenir les spécifications électriques et fonctionnelles officielles de chaque puce en utilisant la documentation disponible publiquement sans interagir physiquement avec l'appareil.

## Objectifs du test
- Localiser les fiches techniques des composants identifiés
- Déterminer les caractéristiques électriques des puces
- Identifier le brochage et la description fonctionnelle

## Comment tester
1. Utiliser les références des puces identifiées lors du test CHSTG-INFO-07.

2. Effectuer des recherches en ligne en combinant la référence de la puce avec le mot-clé :
   - datasheet (fiche technique)

Exemple :
`W25Q128 datasheet`

Exemple de fiche technique :
https://www.winbond.com/resource-files/w25q128jv%20revf%2003272018%20plus.pdf

3. Privilégier les sources officielles du fabricant lorsqu'elles sont disponibles.

4. Archiver les fiches techniques pour l'analyse matérielle ultérieure.

## Remédiation
Non applicable.
