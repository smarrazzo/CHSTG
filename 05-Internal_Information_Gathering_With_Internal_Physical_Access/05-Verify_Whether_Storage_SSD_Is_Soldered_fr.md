# Vérifier si le stockage (SSD) est soudé

|ID          |
|------------|
|CHSTG-INT-05|

## Résumé

Ce contrôle vise à déterminer si le stockage système (SSD) est soudé directement sur la carte mère ou installé sous forme de module amovible. L'objectif est d'évaluer la remplaçabilité physique du dispositif de stockage.

## Objectifs du test
- Identifier si le SSD est soudé à la carte mère
- Déterminer la présence de modules de stockage amovibles
- Évaluer la remplaçabilité physique du stockage

## Comment tester
1. Ouvrir l'appareil selon les procédures de démontage standard.

2. Inspecter visuellement la zone de stockage sur la carte mère.

3. Déterminer si :
   - Les puces de stockage sont soudées directement sur la carte mère
   - Un module de stockage amovible (ex. M.2, SSD SATA) est présent

4. Documenter la configuration de stockage (entièrement soudée ou amovible).

## Remédiation
Non applicable.
