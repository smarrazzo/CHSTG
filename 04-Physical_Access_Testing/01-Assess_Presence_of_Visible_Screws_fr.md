# Évaluer la présence de vis visibles

|ID          |
|------------|
|CHSTG-PHY-01|

## Résumé

Ce contrôle vise à déterminer si les vis de démontage de l'appareil sont visibles de l'extérieur et si elles utilisent des types standard (ex. Phillips M2.5). L'objectif est d'évaluer la facilité d'accès physique sur la base d'une inspection visuelle uniquement.

## Objectifs du test
- Identifier la présence de vis visibles de l'extérieur
- Déterminer si les vis sont standard ou spécialisées
- Évaluer la facilité de démontage selon le type de vis

## Comment tester
1. Effectuer une inspection visuelle de l'extérieur de l'appareil.

2. Identifier :
   - Le nombre de vis visibles
   - L'emplacement des vis
   - Le type de tête de vis (ex. Phillips, Torx, propriétaire)

3. Déterminer si les vis sont :
   - Standard et couramment disponibles (ex. Phillips M2.5)
   - Spécialisées ou peu courantes

4. Documenter le type et l'accessibilité des vis.

## Remédiation
Aucune remédiation véritable n'existe. L'application de scellés inviolables sur les vis peut fournir une indication visuelle de démontage mais n'empêche pas l'accès physique.
