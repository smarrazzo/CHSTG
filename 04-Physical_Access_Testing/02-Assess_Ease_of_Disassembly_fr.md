# Évaluer la facilité de démontage

|ID          |
|------------|
|CHSTG-PHY-02|

## Résumé

Ce contrôle vise à évaluer si l'appareil peut être facilement démonté sur la base d'une observation externe. L'objectif est d'évaluer la simplicité du processus de démontage, notamment l'absence de vis cachées, l'absence d'outils spécialisés et si le démontage pourrait s'effectuer sans causer de dommages visibles.

## Objectifs du test
- Déterminer si le démontage semble simple
- Identifier la présence de vis cachées
- Identifier l'exigence d'outils spécialisés
- Évaluer si le démontage pourrait s'effectuer sans dommage visible

## Comment tester
1. Effectuer une inspection visuelle détaillée de l'extérieur de l'appareil.

2. Identifier :
   - La présence de vis visibles
   - Les vis éventuellement cachées (sous pieds en caoutchouc, étiquettes ou capots)
   - Le type de têtes de vis (standard ou propriétaire)

3. Évaluer si le démontage nécessiterait vraisemblablement :
   - Uniquement des outils standard
   - Des outils spécialisés
   - Des actions destructives (casser des clips ou scellés)

4. Évaluer si la conception de l'appareil semble permettre un démontage sans dommage visible.

5. Documenter la facilité globale de démontage.

## Remédiation
Aucune remédiation véritable n'existe. L'application de scellés inviolables sur les vis peut fournir une indication visuelle de démontage mais n'empêche pas l'accès physique.
